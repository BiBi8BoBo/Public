SILENTTRINITY是一个现代化、异步操作、可操作的c2框架，使用.NET的API调用.NET程序。
其运行方式如下

Installation
#~ git clone https://github.com/byt3bl33d3r/SILENTTRINITY#~ pip3 install --user pipenv && pipenv install && pipenv shell#~ python st.py

Basic Usage
SILENTTRINITY是基于C/S模式来运行的。我们可以用下面的方法来开启服务。
python3 st teamserver <teamserver_ip> <teamserver_password>

然后链接到服务端即可（这里的用户名随意）
python3 st client wss://<username>:<teamserver_password>@<teamserver_ip>:5000
或者使用终端的方式也可以链接到服务端
登录到服务端以后，我们可以使用help命令来查看工具的基础功能：

  • help：查看可使用的命令

  • runrcfile ：运行一个资源文件

  • listeners ：监听器

  • sessions：会话

  • modules：模块

  • stagers：阶段器

  • teamservers：团队服务

监听器
我们首先需要开启一个监听：
[1] ST ≫ listeners                                                              [1] ST (listeners)(http) ≫ list  [1] ST (listeners)(http) ≫ options

里面提供了http、https两种监听方式，可以像msf一样使用set命令来设置地址、端口等，设置后使用start即可开启监听。开启之后便可以在list中看到你开启的监听了，服务端也会有相应的提示。
